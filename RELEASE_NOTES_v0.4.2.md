# Collection 0.4.2

`0.4.2` is a journal UI refinement release built on top of the `0.4.1` hotfix.

## Improvements

- Reworked the Collector Journal into a handbook-style layout with a left tab list and a dedicated reading page.
- Removed the dimmed full-screen overlay when opening the journal so the interface reads cleanly in-game.
- Restyled navigation into journal-like tabs instead of default Minecraft buttons.
- Tightened text layout for Chinese and English lines so clues and progress details stay inside the page frame.
