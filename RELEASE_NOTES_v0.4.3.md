# Collection 0.4.3

`0.4.3` is a focused Collector Journal UI hotfix.

## Fixes

- Disabled Minecraft's default blurred screen background for the Collector Journal.
- Fixed detail-page content overlap near the bottom of the journal.
- Added clipping to the detail card list so long clue text stays inside the page.

- `collection-0.4.3.jar`
