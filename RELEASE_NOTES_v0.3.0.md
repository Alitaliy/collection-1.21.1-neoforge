# Collection 0.3.0

`0.3.0` 是把“下一步”里的功能型目标正式做完的一版：自定义发掘点不再停留在 Feature 原型，收藏家手册也从聊天栏输出升级成了真正的 GUI 与地图线索系统。

## 0.3.0 亮点

### 结构遗迹升级

- `buried_coin_ruin`、`badlands_arrowhead_site`、`shoreline_relic_cache` 全部升级为：
  - 原版结构系统可识别的 `structure`
  - 可定位的 `structure_set`
  - 可扩展的 `template_pool`
  - 真实 `.nbt` 模板遗迹

### 收藏家手册升级

- 普通右键打开分页 GUI 手册。
- 可查看总进度、套组进度、缺失条目和对应线索。
- 潜行右键可领取与未完成套组对应的线索地图。

### 成就与反馈

- 新增首次发现成就。
- 新增三条套组完成成就。
- 新增总收藏完成成就。
- 玩家收藏进度现在同步到客户端，GUI 与地图反馈保持一致。

### 素材策略

- 本版仍复用原版素材。
- 独立美术资源被保留为后续版本统一补做的工作。

## 附件

- `collection-0.3.0.jar`
