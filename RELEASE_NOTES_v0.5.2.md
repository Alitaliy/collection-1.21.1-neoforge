# Collection 0.5.2

`0.5.2` improves the novice Collector trading loop.

## Trading

- Level 1 Collectors now support three early-game trade options:
  - Fixed Collector's Journal sale
  - Random mod collectible recycling trade
  - Random vanilla archaeology item recycling trade
- Vanilla archaeology recycling accepts a curated pool of common brush finds and pottery sherds.
- Existing novice Collectors are patched when players interact with them, allowing older villagers to receive the missing recycling options.
- Level 2-5 Collectors now use randomized trade pools:
  - One random clue map is added on each level-up.
  - One random archaeology recycling trade is added on each level-up.
  - Recycling trades can request mod collectibles or vanilla archaeology finds.
  - Expert and master Collectors can request rarer vanilla archaeology finds such as relic music discs or sniffer eggs.
