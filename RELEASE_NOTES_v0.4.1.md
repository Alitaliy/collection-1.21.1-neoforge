# Collection 0.4.1

`0.4.1` is a focused hotfix release for the `0.4.0` content update.

## Fixes

- Repaired all custom structure templates so `collection:buried_coin_ruin`, `collection:badlands_arrowhead_site`, `collection:shoreline_relic_cache`, `collection:frozen_fossil_site`, and `collection:jungle_idol_shrine` now place blocks correctly.
- Regenerated every shipped `.nbt` structure template with the correct NBT list format required by Minecraft 1.21.1.
- Fixed the Collector Journal screen colors so the page background, text, and status labels render visibly in-game.

## Notes

- Existing empty structure starts generated by `0.4.0` will not backfill automatically.
- Test natural generation in unexplored chunks or a new world after updating.
