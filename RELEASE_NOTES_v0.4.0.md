# Collection 0.4.0

`0.4.0` 把 0.3.0 之后的扩展内容继续往前推进了一步：这次不是单纯再加几个物品，而是把“更多收藏线 + 更多遗迹 + 轮换线索 + 阶段奖励”整合成了一轮新的玩法扩展。

## 0.4.0 亮点

### 新收藏线

- 新增 `化石` 套组
- 新增 `神像` 套组

两条新收藏线都继续复用原版素材，后续再统一替换成独立美术资源。

### 新遗迹

- `collection:frozen_fossil_site`
- `collection:jungle_idol_shrine`

这两类遗迹和此前的自定义发掘点一样，已经完整接入：

- `structure`
- `structure_set`
- `template_pool`
- `.nbt` 模板

### 轮换线索

- 手册现在会按天轮换“今日线索”套组。
- 潜行使用手册时，会优先发放当前轮值套组的线索地图。
- 如果今日线索已经完成，则会自动回退到最近未完成的套组。

### 阶段奖励

- 每个套组在 `1/3` 进度时都会发放一次野外补给。
- 每个套组在 `2/3` 进度时都会发放一次收藏家报酬。
- 如果某个套组在“今日线索”轮值期间完成，还会额外发放一笔绿宝石奖励。
