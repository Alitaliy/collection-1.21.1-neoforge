# Collection 0.2.0

本次 GitHub Release 同时保留 `0.1.0` 与 `0.2.0` 两个版本阶段，便于直接查看项目从首个古币原型到收藏家玩法闭环的演进。

## 0.2.0

### 亮点

- 新增 `收藏家手册`，可查看套组进度与详细线索。
- 新增两大收藏类别：`箭头`、`遗珍`。
- 新增三套完成奖励：`古币展示盒`、`箭头展示盒`、`遗珍名录`。
- 新增玩家持久化收藏进度，死亡后保留。
- 新增两类自定义发掘点：`badlands_arrowhead_site`、`shoreline_relic_cache`。
- 将 `buried_coin_ruin` 升级为多模板发掘点。

### 兼容与分布调整

- 沙漠系考古点更偏向古币与部分遗珍。
- 海洋废墟更偏向遗珍。
- 古迹废墟更偏向箭头，并混入少量跨类别稀有收藏品。

### 附件

- `collection-0.2.0.jar`

## 0.1.0

### 首个可发布版本

- 新增三枚古币收藏品：`1792 Quarter`、`1792 Nickel`、`1789 Penny`。
- 接入 6 类原版考古地点的定向考古掉落替换。
- 新增首个自定义沙漠发掘点 `buried_coin_ruin`。
- 实现专用 archaeology 全局掉落修改器，保证单个可疑方块仅产出一件结果。

### 附件

- `collection-0.1.0.jar`

## 说明

- 当前源码仓库对应 `0.2.0` 状态。
- `0.1.0` 作为历史发布阶段，在本次 Release 中保留说明与二进制文件，便于回溯首个里程碑内容。
