# Collection 0.4.4

`0.4.4` rebuilds the Collector Journal into a two-page handbook layout.

## Improvements

- Replaced the previous panel layout with a book-style interface inspired by classic mod guide books.
- Added a left page for chapter summary, reward preview, and progress.
- Added a right page for chapter navigation or collectible entries.
- Moved long clue text into hover tooltips so page content stays inside the frame.
- Added hand-drawn GUI elements for the book frame, spine, page corners, and page arrows.
